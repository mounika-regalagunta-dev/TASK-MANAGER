Why do we use the Inspect tool in the browser?

We use the Inspect tool to see the code behind a webpage. The browser uses HTML, CSS, and JavaScript to show the webpage, and the Inspect tool lets us look at that code to understand or fix any issues.

What are Unclosed Tags?

Unclosed tags (also called self-closing tags) are HTML tags that do not need a closing tag.

Example:< br>,< img>

What is semantic tags?

Semantic tags are meaningful HTML tags that describe the purpose of the content and help structure a webpage without adding css styles.

,< head >,< body >,< strong >

What is Non- semantic tags?

Non-semantic tags are HTML tags that do not describe the content they contain; they are used only for layout or styling purposes.

Example:< span>,< div >,< b>

What is an Attribute?

An attribute provides additional information about an HTML element.

Example: < href>,< src>
